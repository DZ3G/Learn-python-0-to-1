# Learn Python — 0 to 1

Hey! This is my personal Python learning repo. I started with zero programming experience and I'm documenting everything I learn along the way — notes, exercises, and projects.

Feel free to use any of this as a reference if you're learning too.

## What's in here

### 📁 [`Python for complete beginners`](./Python%20for%20complete%20beginners)
Covers all the fundamentals of Python, following the [Tech With Tim beginner series](https://www.techwithtim.net/tutorials/python-programming).

Topics covered: variables, operators, conditions, loops, lists, functions, file I/O, error handling, classes, and more.

**Final project — CLI ATM** (`Python for complete beginners/final project/`)  
A command-line app that simulates a basic ATM. You can add income and expenses, check your balance, and all transactions are saved and reloaded between sessions.
